**Theorem.** The set of factorions in $\mathbb{N}$ is exactly $\{1, 2, 145, 40585\}$.

*Proof.* We show the two inclusions separately.

$(\subseteq)$ Let $n \in \mathbb{N}$ be a factorion. By **Factorion Upper Bound**, every factorion satisfies $n \leq 2540160$. Applying **Factorion Finite Check** to $n$ with this bound, membership in the finite candidate set is decidable, and $n$ must lie in $\{1, 2, 145, 40585\}$.

$(\supseteq)$ Let $n \in \{1, 2, 145, 40585\}$. In each case one verifies directly that $n \leq 2540160$, so **Factorion Finite Check** applies, and one confirms by direct computation that each element is indeed a factorion. $\blacksquare$
